Each broadcaster site gets its own registration key, valid only for the channel ids its scopes match — a leaked key can no longer touch other sites' channels, and revoking one site is a status flip here instead of re-keying everyone. The secret is shown once at enrollment; put it in that site's broadcaster .env as PUBLISHER_KEY together with PUBLISHER_NAME. Registry lives in the panel-private secrets/publishers.json.
